🎮 使用指南

基本操作：

  1. 选择正确的游戏版本
  2. 启动游戏，修改器会自动检测并连接
  3. 设置想要的阳光/金币数值
  4. 勾选"无限"选项启用功能
  5. 勾选"自动锁定"保持数值恒定

新功能使用：

  1. 多版本支持: 在顶部选择您游戏的正确版本
  2. 自动检测: 游戏启动后修改器会自动连接，无需手动操作
  3. 历史记录: 在"历史记录"选项卡查看所有修改操作
  4. 地址扫描: 点击"扫描地址"按钮手动刷新游戏地址

默认热键参考：

  • 修改阳光: Ctrl + S
  • 切换无限阳光: Ctrl + Shift + S
  • 修改金币: Ctrl + C
  • 切换无限金币: Ctrl + Shift + C
  • 刷新游戏: F5
